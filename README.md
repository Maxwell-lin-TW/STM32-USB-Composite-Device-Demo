# STM32-USB-Composite-Device-Demo
STM32 USB composite device porting project using ST USB Device middleware, supporting CDC + HID with custom descriptors and host-side test notes.

# Hardware 

Base on following evaluation boards:

[NUCLEO-C071RB](https://www.st.com/en/evaluation-tools/nucleo-c071rb.html)
[NUCLEO-H753ZI](https://www.st.com/en/evaluation-tools/nucleo-h753zi.html)
[DRG_ST_10-STM32-G474-VET6MINI](https://gitee.com/gaoyanzeng/drg_st_10_-stm32-g474-vet6mini)


# Firmware
